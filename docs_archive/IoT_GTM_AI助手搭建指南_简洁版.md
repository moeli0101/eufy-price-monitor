## 2. 知识库机制说明

### 小moe的两个知识库

小moe同时使用两个知识库：

#### A. bot.py本地知识库（自动化）

```
飞书多维表格（文档索引）
         ↓
@小moe 更新知识库（一句话）
         ↓
bot.py自动读取所有文档
         ↓
生成 knowledge_base_generated.py
         ↓
小moe立即使用 ✅
```

**特点**：
- ✅ 全自动，一句话完成
- ✅ 速度快（本地读取）
- ✅ 适合快速问答

#### B. AIME云端知识库（可选）

```
飞书文档
    ↓
选择1：直接粘贴到AIME平台 ← 手动
选择2：本地CC批量提炼 → 粘贴摘要 ← 辅助
    ↓
AIME智能体使用
```

**特点**：
- ⭐ 可选优化
- ⭐ 适合深度分析
- ⭐ 需要手动维护

### 本地CC的作用

**不是替代，是辅助**：
- 当有50+篇文档时，本地CC可以批量提炼关键信息
- 用户可以选择性地将摘要粘贴到AIME云端
- 但对于日常使用，bot.py的自动更新已经足够

## 4. 启动bot.py（10分钟）

### 4.1 安装依赖

```bash
pip3 install lark-oapi requests urllib3 schedule feedparser
```

### 4.2 配置bot.py

编辑 `bot.py` 开头：

```python
FEISHU_APP_ID = "你的App ID"
FEISHU_APP_SECRET = "你的App Secret"
AIME_API_KEY = "你的AIME Key"
AIME_API_URL = "你的AIME URL"
BITABLE_APP_TOKEN = "你的app_token"  # 可选
```

### 4.3 创建飞书多维表格（知识库索引）

1. 新建多维表格「小moe知识库索引」
2. 创建3列：

| 列名 | 类型 | 说明 |
|------|------|------|
| 分类 | 单选 | 竞品分析/市场策略/产品规划 |
| 文档内容 | 文本 | 文档标题 |
| 链接 | 超链接 | @飞书文档 |

3. 填入文档：

| 分类 | 文档内容 | 链接 |
|------|----------|------|
| 竞品分析 | 2026 Q1竞品分析 | @文档 |
| 市场策略 | 北美GTM策略 | @文档 |
| ... | ... | ... |

4. 获取 app_token：

```
https://anker-in.feishu.cn/base/XYZ123abc456
                                  ↑
                            这就是app_token
```

填到 bot.py：
```python
BITABLE_APP_TOKEN = "XYZ123abc456"
```

### 4.4 启动机器人

```bash
cd ~/Desktop/feishu_bot
python3 bot.py
```

**成功标志**：
```
[知识库] 成功导入生成的知识库，共 16 篇文档
[产品库] 已加载 125 个产品
机器人已启动! 等待消息...
[INFO] connected to wss://msg-frontier.feishu.cn
```

### 4.5 后台运行

```bash
nohup python3 bot.py > bot_debug.log 2>&1 &
```

### CC本地版（个人使用）

**安装**：
```bash
npm install -g @anthropic-ai/claude-code

# 配置飞书MCP
npx -y @larksuiteoapi/lark-mcp login -a YOUR_APP_ID -s YOUR_APP_SECRET

claude mcp add-json lark-mcp --scope=user '{
  "command":"npx",
  "args":["-y","@larksuiteoapi/lark-mcp","mcp",
  "-a","YOUR_APP_ID","-s","YOUR_APP_SECRET",
  "--oauth","--token-mode","user_access_token"]
}'
```

**适合**：
- 批量处理文档（50+篇）
- 数据抓取和分析
- 自动化工具开发
- Vibe Coding

## 3. 常见问题FAQ

### Q1：小moe不回复？

**检查**：
```bash
ps aux | grep bot.py  # 是否运行
tail -f bot_debug.log  # 查看日志
```

**解决**：
```bash
pkill -9 -f "bot.py"
python3 bot.py
```

### Q2：知识库回答不准确？

**步骤**：
1. 检查系统提示词是否有 `{{#context#}}`
2. 在飞书群发：`@小moe 更新知识库`
3. 等待回复"✅ 知识库更新完成"
4. 重新提问

### Q3：如何添加新文档到知识库？

**方法1（推荐）**：
1. 在多维表格新增一行
2. @文档链接
3. `@小moe 更新知识库`

**方法2（可选）**：
1. 用本地CC批量提炼
2. 手动粘贴到AIME平台

### Q4：本地CC连不上飞书？

```bash
# 重新登录
npx -y @larksuiteoapi/lark-mcp login \
  -a YOUR_APP_ID -s YOUR_APP_SECRET

# 检查权限
# 确认飞书应用有 docx:document 权限
```

### Q5：如何停止/重启？

```bash
# 停止
pkill -9 -f "bot.py"

# 启动
cd ~/Desktop/feishu_bot
nohup python3 bot.py > bot_debug.log 2>&1 &

# 查看状态
ps aux | grep bot.py
```
