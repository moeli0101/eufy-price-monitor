## 🚀 传统方式 vs 简化方式

### 传统方式（45分钟）
```
1. 手动修改bot.py配置（复制粘贴5个凭证）

```

## 📋 前置准备（必须手动做的）

这些必须在网页上操作，无法自动化：

### 1. 飞书机器人（10分钟）
- 访问 https://open.feishu.cn/
- 创建应用「小moe」
- 开通权限：`im:message`, `im:message.group_at_msg`, `im:chat`, `bitable:app`, `docx:document`
- 获取：**App ID** 和 **App Secret**

### 2. AIME智能体（5分钟）
- 访问 https://ai.anker-in.com/
- 创建智能体「xiaomoe」
- **先别填系统提示词**（setup脚本会自动生成）
- 获取：**API Key** 和 **API URL**

## ⚡ 一键配置（5分钟）

### 运行配置脚本

```bash
cd ~/Desktop/feishu_bot
python3 setup_xiaomoe.py
```

脚本会自动：
- ✅ 收集你的凭证（交互式输入）

## 📝 完成配置（2分钟）

### 1. 复制系统提示词到AIME

```bash
# 脚本已生成文件
cat aime_system_prompt.txt

# 复制全部内容 → 粘贴到AIME「系统提示词」框
# ⚠️ 必须包含 {{#context#}} 标签！
```

### 2. 创建多维表格（手动）

在飞书新建多维表格「小moe知识库索引」，3列：

| 列名 | 类型 | 选项 |
|------|------|------|
| 分类 | 单选 | 竞品分析/市场策略/产品规划/区域数据 |
| 文档内容 | 文本 | - |
| 链接 | 超链接 | @飞书文档 |

获取 `app_token`：URL中 `/base/` 后面那串

### 3. 启动机器人

```bash
# 方式1：前台运行（看日志）
./start_bot.sh

# 方式2：后台运行（持久化）
nohup python3 bot.py > bot_debug.log 2>&1 &
```
