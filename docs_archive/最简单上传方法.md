# 🚀 最简单的上传方法（3分钟）

## 方案1：使用GitHub网页直接上传（推荐）

1. 打开你的仓库：https://github.com/moeli0101/eufy-price-monitor

2. 点击 **uploading an existing file**（上传现有文件）

3. 把这些文件拖进去：
   - 📁 整个 `docs` 文件夹
   - 📁 整个 `.github` 文件夹
   - 📄 `README.md`
   - 📄 `price_dashboard_v5.html`
   - 📄 `daily_price_refresh.py`
   - 📄 `.gitignore`

4. 在底部 "Commit changes" 输入：`Initial commit`

5. 点击 **Commit changes**

完成！✅

---

## 方案2：使用GitHub Desktop（更简单）

1. 下载安装：https://desktop.github.com

2. 打开后登录你的GitHub账号

3. 点击：**Add** → **Add existing repository**

4. 选择文件夹：`/Users/anker/Desktop/feishu_bot`

5. 点击：**Publish repository**

完成！✅

---

## 下一步：启用GitHub Pages

上传完成后：

1. 打开：https://github.com/moeli0101/eufy-price-monitor/settings/pages

2. **Source** 下面：
   - Branch: 选择 `main`
   - Folder: 选择 `/docs`

3. 点击 **Save**
