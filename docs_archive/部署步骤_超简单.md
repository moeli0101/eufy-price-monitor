# 🚀 部署到GitHub Pages - 3分钟搞定！

## 步骤1: 创建GitHub仓库（1分钟）

1. 打开 https://github.com/new
2. 填写信息：
   - **Repository name**: `eufy-price-monitor`（必须用这个名字）
   - **Description**: `eufy澳洲价格监控系统`
   - 选择 **Public**（公开）
   - ❌ **不要**勾选 "Add a README file"
   - ❌ **不要**勾选其他选项
3. 点击 **Create repository**

## 步骤2: 上传代码（复制粘贴）（1分钟）

创建仓库后，GitHub会显示一个页面，找到 **"...or push an existing repository from the command line"** 这一段。

然后在你的终端（我这里）运行：

```bash
cd /Users/anker/Desktop/feishu_bot
git remote add origin https://github.com/moeli0101/eufy-price-monitor.git
git branch -M main
git push -u origin main
```

## 步骤3: 启用GitHub Pages（1分钟）

1. 在你的仓库页面，点击 **Settings**（设置）
2. 左侧菜单找到 **Pages**
3. 在 **Source** 下：
   - Branch: 选择 **main**
   - Folder: 选择 **/docs**
4. 点击 **Save**

## 🎉 完成！

等待1-2分钟后，你会看到：

```
✅ Your site is published at:
https://moeli0101.github.io/eufy-price-monitor/
```

**这就是你的永久网址！**
